Detailed Description

template<typename TUnit, int d>
class Vector< TUnit, d, std::enable_if_t< IsFloatSpecialization< TUnit, d > > >

Specialization for floats or units of float precision. Copies interface of generic Vector type, but uses SSE instruction to optimize operations.
