Detailed Description

template<class TDerived>
class Kernel< TDerived >

A base class for all SPH kernels, providing interface for computing kernel values and gradients. All derived class must implement method value and grad. Both function take SQUARED value of dimensionless distance q as a parameter. Function value returns the kernel value, grad returns gradient DIVIDED BY q.
