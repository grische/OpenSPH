Detailed Description

template<typename TScalarRng, typename TPdf, typename TJacobian>
class VectorPdfRng< TScalarRng, TPdf, TJacobian >

Generic generator of random vectors using sampling with given PDF. PDF must have units such that pdf(x) jacobian(x) dV = dimensionless, however it does NOT have to be normalized (integral does not have to be 1)
